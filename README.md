# 使用Docker部署Django项目

## Quickstart

```bash
$ git clone git@github.com:anshengme/docker-deploy-django-example.git
$ cd docker-deploy-django-example
$ docker-compose up
```

打开浏览器访问：`http://127.0.0.1:9999`
